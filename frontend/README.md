# LeafCard Frontend - 轻羽卡管系统前端

LeafCard 是一个企业级卡密管理系统前端项目，基于 Vue 3 和 Element Plus 构建。

## 项目特性
- 现代化的用户界面设计
- 响应式布局，支持多设备访问
- 基于 Vue 3 的组件化开发
- 使用 Element Plus UI 组件库
- 支持 PWA 渐进式 Web 应用

## 项目设置
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
